# Stewart-prealpha
Versión de concepto de Stewart. Conoce sobre el proyecto completo en https://linktr.ee/Stewart_devteam

Esta versión es solo una demostración y no busca representar de forma fiel el juego completo. En cambio es más un experimento para dar comienzo al desarrollo del juego de forma temporal, antes de crear una versión completa.

Si esta versión adquiere forma, se volverá la base para el proyecto definitivo.
